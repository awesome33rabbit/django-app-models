# Cookiecutter Django app

使用`cookiecutter`创建`django app`



# Features

- Django 3+
- Python 3.9



# Usage

安装依赖：

```
pip install cookiecutter
```

使用脚本`main.py`：

```
# 下载脚本`main.py`，运行即可
python main.py
```
